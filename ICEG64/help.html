Help

Offical internet help page for the ICEG64 project. Download proect here!
Created by Andor Csete

Function keywords:
  • builtin - open built-in command menu
  • custom - open custom code editor
  • help - open help page in console
  • webhelp - open help page in browser (This page)


Built-in command keywords:
Keyword Function description
key create_key() Generate random binary characters. (not in use)
saltB create_saltBin() Generate index length random binary
saltI create_saltInt() Generate index length random integer
saltS create_saltStr() Generate index length random string
e convertStB() convert string to binary, insert index length salt at the beginning
d convertSfB() convert string from binary, cut index length salt from the beginning
eI convertStI() convert string to integer, insert index length salt at the beginning
dI convertSfI() convert string from integer, cut index length salt from the beginning
_NONE_ randomNum() generate a pseudo-random number, based on time
encode encode() Convert string to integer, multiply it by 29, and translate it, than insert index length salt
decode decode() Cut index lenght salt off, divide by 29 convert to string
transNew random_translate() Create a random translate table (dictionary), with a pattern using upper/lowercase, numbers, and special charachters
transT translate() Translate string to a coded text using a constant dictionary (generated random before)
transF translate_back() Translate coded text to string using a constant dictionary (generated random before)
ICE ICEG64 Translate string, convert it to integer, translate it again, insert index lenght salt
deICE deICEG64() Remove salt, transalte back, convert integer to string, translate it again
ICEfib ICEG64Fib() [NOT FINISHED YET!] translate it, convert it to integer, translate it, insert index length salt every fibonacci position
deICEfib deICEG64Fib() [NOT FINISHED YET!] Remove index lenght salt from every fibonacci position, transalate it back, convert string form integer, translate it back
_NONE_ generate_fibonacci() Generate fibonacci numbers that are below X. Created for ICEG64fib
ICE32 ICEG32() Lighter version of ICEG64. Translate string, than convert it to integer. Insert index length salt to the beginning.
deICE32 generate_fibonacci() Decode ICEG32. Remove salt, convert integer to string, translate it back.
q _NONE_ Exit built-in mode to function selecter.
